# Pritunl Ubuntu 22.04 Installer

`install_pritunl_ubuntu2204.sh` installs MongoDB, OpenVPN and Pritunl on Ubuntu 22.04.

Example:

```bash
curl -fsSL <RAW_URL>/install_pritunl_ubuntu2204.sh -o install_pritunl_ubuntu2204.sh
chmod +x install_pritunl_ubuntu2204.sh
sudo ./install_pritunl_ubuntu2204.sh --public-address 114.134.185.4
```
